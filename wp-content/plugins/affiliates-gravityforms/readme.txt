1) Installation
==============================================================================

    Affiliates Gravity Forms integrates the Affiliates plugins with
    Gravity Forms - Gravity Forms is required for this extension to work.
    This extension also requires one of the Affiliates plugins, the free
    Affiliates plugin or one of the premium versions, Affiliates Pro or
    Affiliates Enterprise.

    1) Backup
    --------------------------------------------------------------------------

    Back up your WordPress installation files and database.

    2) Install and activate Affiliates, Affiliates Pro or Enterprise
    --------------------------------------------------------------------------

    If you have not done so already, install and activate one of the
    Affiliates plugins.

    * To install the free Affiliates plugin automatically:

    Log in on your site with your administrator account. From your Dashboard,
    go to "Plugins > Add New", input "Affiliates" (without the quotes) in the
    search field located besides the "Search Plugins" button, click the button
    and locate the Affiliates plugin by itthinx in the results.

    Click the "Install Now" link for that plugin and confirm that you want to
    install the plugin.

    Once the plugin has been downloaded and installed, confirm that you want
    to activate it.

    * To install the premium Affiliates Pro or Affiliates Enterprise manually:

    Visit http://www.itthinx.com/plugins/affiliates-pro
    Once you have obtained the plugin, go to "Plugins > Add New > Upload"
    and click the "Browse..." button to select the plugin's zip file and
    click on the "Install Now" button.

    After you have installed the plugin, proceed to activate it and follow
    the documentation for setup.

    3) Install and activate the Affiliates Gravity Forms plugin
    --------------------------------------------------------------------------

	Go to Affiliates > Settings > Integrations, install and activate the
	integration for Gravity Forms.

2) Setup
==============================================================================

    1) Make sure that you have followed the installation instructions above.
    2) Forms have a new settings section under the Form Settings tab, you will
       find the Affiliates subsection. Adjust the settings for each form
       as required.

3) Update
==============================================================================

    If you are notified of an updated version of the plugin, follow these
    instructions to update it:

    1) Backup
    --------------------------------------------------------------------------

    Back up your WordPress installation files and database.

    2) Upgrade to the new release
    --------------------------------------------------------------------------

    Upgrading requires an active service key and the itthinx updates plugin
    installed. Once a new version is available, you will see a notification
    in your WordPress Dashboard where you can have the plugin updated.
